Scripts Python de Montrouge

Protocole pour l'utilisation des scripts

  1. Télécharger l'archive à l'adresse suivante : Lien de téléchargement
  2. Décompresser *l'intégralité* du dossier sur l'ordinateur
  3. Les scripts sont dans le sous-dossier programmes_lecons.

Remarque : Afin d'améliorer la lisibilité des scripts, et de faciliter l'écriture de nouveaux scripts, les fonctions permettant l'affichage correct sont écrits dans des modules séparés. Pour cette raison, il est nécessaire de télécharger toute l'archive.
